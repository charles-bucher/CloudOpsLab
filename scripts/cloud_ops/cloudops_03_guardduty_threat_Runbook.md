## Overview
Enables Amazon GuardDuty across accounts and regions to ensure continuous threat detection.

## When to Use
- Initial cloud environment setup
- Security hardening
- Compliance preparation (SOC2, ISO27001, CIS)

## Prerequisites
- AWS permissions: `guardduty:*`, `organizations:ListAccounts`
- GuardDuty must be supported in the region
